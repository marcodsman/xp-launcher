# Kill Bill (OpenBOR)

Freeware OpenBOR pak. Own launcher entry / own folder: `C:\XP_Share\openbor-killbill`
= our 2011 SDL 1.2 `OpenBOR.exe` + `Paks/DC1.PAK` (the pak is 2005-era, runs on the
old XP-safe engine). The pak is game data (not repo-tracked, like BOR.PAK); only the
config lives here.

Pad + fullscreen fixed exactly like Beats of Rage (see memory `openbor-borcfg-format`):
`Saves/DC1.cfg` patched via `patch_borcfg.py` — P1=joy1, P2=joy0, d-pad hat -> move,
PSX-diamond faces, `screen[0]={2,0}` -> 640x480 landscape, `fullscreen=1`.

Source pak: archive.org/details/kill-bill.-7z (KillBill.zip -> Paks/DC1.PAK).
